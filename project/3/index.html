智慧環境監測系統

你是透過 QR Code 來到這裡的。此頁為 mk-chou.com/project/3 的範例內容，之後可替換成你的實際專題。

專題簡介

本專題使用 ESP32 微控制器，搭配 DHT22 溫溼度感測器與 BH1750 光照感測器， 即時收集環境數據並上傳至雲端平台，方便遠端監控與分析。

主要功能

  • 即時溫度、濕度、光照強度監測
  • Wi-Fi 連線，自動上傳資料
  • 網頁儀表板顯示歷史趨勢
  • QR Code 快速分享專題成果

使用技術

  • ESP32
  • Arduino
  • MQTT
  • HTML / CSS
